git的使用

使用git，目前本人所知的有三种方法：

1)github for windows
可以在github官网上下载，图形化操作。
基本操作还是比较简单的。但不推荐，推荐用第2种。

2)git bash（推荐）
使用命令操作，更简洁暴力，但需要学习一下。
首先在官网下载安装。
常用命令：
从远程仓库上获得：git clone ...
查看状态：git status
查看区别：git diff
准备把修改commit到本地仓库：git add .
把修改commit到本地仓库：git commit -m "这里写注释"
把修改push到远程仓库：git push
从远程仓库获取变化：git pull

3)eclipse插件
可以从github上import项目到本地，在eclipse中导入成项目，修改之后提交。
具体eclipse的team菜单中的命令，参考上面2)中使用git bash时的命令。

另外，git@osc是中国版的github，好处在于私有库免费，公司用户想有私有库，又不想花钱的，可以考虑使用。

建立新库的推荐操作：先在github网站上建立仓库，clone到本地文件夹，在该文件夹中初始化项目，再push上去。

如果是已有的项目，则可以team-share project创建本地库（建议在父文件夹中建库，否则项目会被从workspace中挪走），之后commit，再push。
