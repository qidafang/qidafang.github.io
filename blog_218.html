linux学习

linux一直是个人比较惧怕的东西，太大太陌生，不过据说其实稍微学学也就够用了……
看了慕课网“Linux Guide for Developers”课程，
然后跟着老师的推荐大概看了看《The Linux Command Line》中文版。
对基本操作有点信心了，以后随着用再深入。
下面是笔记——


1.跳转
history:查看历史命令 可以history | less这样分页看
cd:用tab补齐，双击tab看多个备选
pwd:当前在哪儿
ls:看当前目录下文件 -a可以看隐藏文件（以.开头的文件） -l可以看细节
file:看文件类型
less:查看文件

2.文件&目录
1)基本
cp f1 f2 拷贝文件
cp -r d1 d2 拷贝文件夹
mv f1 f2 重命名文件
mv f1 d1 移动文件
mv d1 d2 重命名文件夹
rm f 删除文件
rm -r d 删除文件夹
创建文件：用编辑器保存
mkdir d 创建文件夹
ln file link 创建链接
2)压缩解压
unzip/zip
tar zxvf/tar zcvf
tar jxvf/tar jcvf
分别对付zip/tar.gz/tar.bz2文件，打包目录后面参数1为文件名，参数2为目录名
3)命令
type显示命令类型
which显示程序位置
help显示命令帮助
man显示命令手册

3.重定向
标准输入stdin，标准输出stdout，标准错误输出stderr
>输出重定向，>>输出追加重定向，单大于号会清空，2>错误输出重定向
<输入重定向，|管道线连接
cat 连接文件
sort 排序
uniq 去重
grep 匹配，语法：grep pattern content
例：ls /bin /usr/bin | sort | less

4.权限
读r写w执行x，
以下字符串每3个一组，依次拥有者、小组、其他人：-rw-rw-r--，表示拥有者的rw权限，小组的rw权限，其他人的r权限
chmod 666 a.txt，666表示8进制的110110110，即rw-rw-rw-
用ls -l a.txt看权限

5.进程
进程号pid
ps aux查看进程
kill 进程号 杀死进程
top 计算机运行状态信息

6.搜索
locate全局搜索文件 
find在目录中搜索文件 
grep搜索字符串

7.网络操作
ssh协议，22端口
我个人用的是xshell和xftp，据说putty也不错
ping，不解释
netstat：网络连接
wget：下载

8.安装软件
个人用yum还得研究

9.脚本编程
老师提到的原则：
*声明语言
*命令即语句
*位置参数
*脚本不在当前shell执行
*循环控制
*远程执行
*确认执行
*用其他语言写脚本，bash晦涩
