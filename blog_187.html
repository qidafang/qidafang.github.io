oracle，好久不见

久违的数据库，久违的业务场景，久违的人。
好像回到了多年前的北三环。
夜深忽梦少年事，梦啼妆泪红阑干。

1.
从oracle官网上下载两个巨大无比的安装包，分别解压后放到一起，安装。
2.
安装PL/SQL，如果装oracle的机器是64位的，还要按这个教程来解决问题。
3.
如果单纯想测试oracle服务是否好用，写个单元测试尝试jdbc连就行了。

    @Test
    public void testOracle() throws Exception {
        Class.forName("oracle.jdbc.driver.OracleDriver");
        Connection c = DriverManager.getConnection("jdbc:oracle:thin:@localhost:1521:orcl", "test", "test");
        c.close();
    }

4.
在这之前当然需要把oracle驱动添加到maven引用。
但oracle驱动maven中央库没有，需要手动添加到本地库：
在oracle首页点downloads，页面中的drivers栏里有jdbc驱动链接，点进去下载。
然后mvn install:install-file -Dfile=ojdbc6.jar -DgroupId=com.oracle -DartifactId=ojdbc6 -Dversion=11.2.0.1.0 -Dpackaging=jar
然后再在项目中引入

<dependencies>
    <dependency>
        <groupId>com.oracle</groupId>
        <artifactId>ojdbc14</artifactId>
        <version>11.2.0.1.0</version>
    </dependency>
</dependencies>
