php基础语法，兼说切换语言的问题

是ruby基础语法和python基础语法的继续，目的是减少切换编程语言时的成本。

一图流：为什么说切换编程语言很容易——

 

一门编程语言，需要考察的是其使用领域、语言基础、最常用的库和框架，其中语言基础包括语法（常量变量运算符，顺序选择循环，数字文本日期，数组和字典，类和对象，源码文件们的组织方式）和核心库（关于文件系统，网络，操作系统等的库），一般来说语法足以用一个短篇博文列举出来，而核心库的手册也不会有太大篇幅（十几个网页足够了）。

而最常用的库和框架，既然是最常用这个级别，其官网的tutorial教程、demo示例、api文档一般都会不错，如果可能的话阅读源码更是加深理解的最好方法。所以这部分虽然知识量会比较大，但也不用担心，至少有问题可以随时查阅嘛。至于资料不足、代码也不那么好读的库和框架，最好谨慎着用它，能不用就不用，否则出来一个奇葩bug影响都会是致命的。

至于其他零零碎碎的细枝末节知识和细节，无视之就好，遇到了现研究是最划算的方式。有的还会随着时间的流逝而失去价值（如css的hack），所以完全不必在这上面下功夫，否则就是强迫症。

至于该语言适用领域的问题，一开始就是该搞懂的，这才不会干出用锤子开啤酒瓶盖的事情（比如php官网说php能用来做桌面程序，但你会这么做吗？连官网自己都不太赞成），否则，你再天纵英才，再悬梁刺股，事情搞砸了，都只有两个字：活该。

 

正文：php基础语法（参考：w3school的教程，另外该教程中的手册部分用作日常参考）：

}

#字符串函数

define("CONST", 1);

}

}

while(false){
    #
}
do{
    #
}while(false);#最后的分号很关键
for($x=1; $x<=10; $x++){
    #
}
foreach($array as $value) {#遍历数组
    #
}
foreach($map as $k=>$v){#遍历关联数组

}

#函数
function myFunction($a){
    #
}

#超全局变量
$g = $GLOBALS["a"];#全局作用域可用

$g = $_POST["a"];#post提交的数据

include 'test2.php';
